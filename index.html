Introducción

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, creado por Guido van Rossum y lanzado por primera vez en 1991. Se destaca por su simplicidad y legibilidad, lo que lo hace ideal tanto para principiantes como para desarrolladores experimentados.

Características principales de Python:

  • Sintaxis clara y legible: Utiliza indentación para delimitar bloques de código, lo que fomenta un estilo de programación limpio y consistente.
  • Tipado dinámico: Las variables no necesitan ser declaradas con un tipo específico antes de usarlas.
  • Interpretado: El código se ejecuta línea por línea, lo que facilita la depuración y el desarrollo iterativo.
  • Multiplataforma: Funciona en diversos sistemas operativos como Windows, macOS y Linux.
  • Multiparadigma: Soporta programación orientada a objetos, programación imperativa y programación funcional.
  • Extensa biblioteca estándar: Incluye módulos y funciones para una amplia gama de tareas, desde procesamiento de texto hasta desarrollo web.
  • Gran comunidad y ecosistema: Cuenta con una comunidad activa y una gran cantidad de bibliotecas de terceros disponibles a través de PyPI (Python Package Index).
  • Versátil: Se utiliza en diversos campos como desarrollo web, ciencia de datos, inteligencia artificial, automatización, scripting y más.

Python se ha convertido en uno de los lenguajes de programación más populares debido a su facilidad de uso, versatilidad y potencia, lo que lo hace adecuado para una amplia gama de aplicaciones y proyectos.

Estructura de datos
  1. Listas:
    • Colección ordenada y mutable de elementos.
    • Pueden contener diferentes tipos de datos.
    • Se definen con corchetes []. Ejemplo:
      • mi_lista = [1, 2, 3, "python", True]
  2. Tuplas:
    • Colección ordenada e inmutable de elementos.
    • Similar a las listas, pero no se pueden modificar después de crearse.
    • Se definen con paréntesis (). Ejemplo:
      • mi_tupla = (1, 2, 3, "python", True)
  3. Diccionarios:
    • Colección no ordenada de pares clave-valor.
    • Las claves deben ser únicas e inmutables.
    • Se definen con llaves {}.Ejemplo:
      • mi_diccionario = {"nombre": "Ana", "edad": 28, "ciudad": "Madrid"}
  4. Conjuntos:
    • Colección no ordenada de elementos únicos.
    • Útiles para eliminar duplicados y realizar operaciones de conjunto.
    • Se definen con llaves {} o la función set(). Ejemplo:
      • mi_conjunto = {1, 2, 3, 4, 5}

    5. Cada una de estas estructuras de datos tiene sus propias características y métodos asociados, lo que las hace adecuadas para diferentes tipos de tareas en programación.

Control de Flujo
  1. Condicionales: Permiten ejecutar diferentes bloques de código según se cumplan ciertas condiciones. Ejemplo:

                    edad = 18
                    if edad >= 18:
                        print("Eres mayor de edad")

                    else:

  2. Bucles:
    1. For: Se usa para iterar sobre una secuencia (lista, tupla, diccionario, conjunto o cadena). Ejemplo:
      2. for i in range(5): print(f"Número: {i}")
    2. While: Ejecuta un bloque de código mientras una condición sea verdadera. Ejemplo:
      3. contador = 0 while contador < 5: print(f"Contador: {contador}") contador += 1
  3. Manejo de excepciones: Permite manejar errores y excepciones que puedan ocurrir durante la ejecución del programa. Ejemplo:
    4. try: resultado = 10 / 0 except ZeroDivisionError: print("Error: División por cero") finally: print("Este bloque siempre se ejecuta")
  4. Declaraciones de control adicionales:
    1. Break: Sale del bucle actual.
    2. Continue: Salta a la siguiente iteración del bucle.
    3. Pass: No hace nada, se usa como marcador de posición.

      4. Ejemplo:

      for i in range(10): if i == 3: continue if i == 8: break print(i)

    5. Estos elementos de control de flujo son fundamentales en Python y permiten crear programas más complejos y flexibles, controlando la ejecución del código según las condiciones y necesidades específicas.

Funciones y módulos
  1. Funciones:
    • Bloques de código reutilizables que realizan una tarea específica.
    • Se definen con la palabra clave 'def'.
    • Pueden tener parámetros de entrada y devolver valores.
    • Tienen un ámbito local para las variables definidas dentro de ellas.
    • Ejemplo de función:
      • def saludar(nombre): return f"Hola, {nombre}!" mensaje = saludar("Ana") print(mensaje) # Imprime: Hola, Ana!
  2. Módulos:
    • Archivos que contienen código Python, incluyendo funciones, clases y variables.
    • Se importan usando la palabra clave 'import'.
    • Pueden ser módulos de la biblioteca estándar, de terceros o propios.
    • Ayudan a mantener el código organizado y modular.
    • Ejemplo de uso de módulos:
      • import math radio = 5 area = math.pi * radio ** 2 print(f"El área del círculo es: {area}") from random import randint numero_aleatorio = randint(1, 10) print(f"Número aleatorio: {numero_aleatorio}")
  3. Beneficios de usar funciones y módulos:
    • Reutilización de código
    • Mejor organización y estructura del programa
    • Facilidad de mantenimiento y depuración
    • Posibilidad de compartir código entre diferentes proyectos
Programación orientada a objetos
  1. Clases:
    • Plantillas para crear objetos que encapsulan datos y comportamientos.
    • Se definen con la palabra clave 'class'.
    • Contienen atributos (variables) y métodos (funciones).
    • El método __init__ se usa para inicializar objetos.
    • Ejemplo de clase:
      • class Perro: def __init__(self, nombre, edad): self.nombre = nombre self.edad = edad def ladrar(self): return f"{self.nombre} dice: ¡Guau!" mi_perro = Perro("Firulais", 3) print(mi_perro.ladrar()) # Imprime: Firulais dice: ¡Guau!
  2. Herencia:
    • Permite crear nuevas clases basadas en clases existentes.
    • La clase hija hereda atributos y métodos de la clase padre.
    • Facilita la reutilización de código y la creación de jerarquías.
    • Ejemplo de herencia:
      • class Animal: def __init__(self, nombre): self.nombre = nombre class Perro(Animal): def ladrar(self): return f"{self.nombre} dice: ¡Guau!" mi_perro = Perro("Firulais") print(mi_perro.ladrar()) # Imprime: Firulais dice: ¡Guau!
  3. Polimorfismo:
    • Permite que objetos de diferentes clases respondan al mismo método.
    • Facilita el uso de una interfaz común para diferentes tipos de objetos.
    • Se implementa a través de herencia y métodos con el mismo nombre.
    • Ejemplo de polimorfismo:
      • class Gato(Animal): def hacer_sonido(self): return f"{self.nombre} dice: ¡Miau!" class Perro(Animal): def hacer_sonido(self): return f"{self.nombre} dice: ¡Guau!" def animal_sonido(animal): print(animal.hacer_sonido()) gato = Gato("Michi") perro = Perro("Firulais") animal_sonido(gato) # Imprime: Michi dice: ¡Miau! animal_sonido(perro) # Imprime: Firulais dice: ¡Guau!
Manejo de archivos / entrada y salida
  1. Apertura y cierre de archivos:
    • Se usa la función 'open()' para abrir archivos.
    • Es importante cerrar los archivos después de usarlos con 'close()'.
    • Se recomienda usar la estructura 'with' para manejar archivos de forma segura.
    • Ejemplo de apertura y cierre de archivo:
      • with open('archivo.txt', 'r') as archivo: contenido = archivo.read() print(contenido)
  2. Lectura de archivos:
    • Método 'read()': Lee todo el contenido del archivo.
    • Método 'readline()': Lee una línea a la vez.
    • Método 'readlines()': Lee todas las líneas y las devuelve como una lista.
    • Ejemplo de lectura de archivo:
      • with open('archivo.txt', 'r') as archivo: for linea in archivo: print(linea.strip())
  3. Escritura en archivos:
    • Modo 'w': Sobrescribe el archivo si existe, o crea uno nuevo.
    • Modo 'a': Añade contenido al final del archivo.
    • Método 'write()': Escribe una cadena en el archivo.
    • Ejemplo de escritura en archivo:
      • with open('nuevo_archivo.txt', 'w') as archivo: archivo.write("Hola, mundo!\n") archivo.write("Esta es una nueva línea.")
  4. Entrada/salida estándar:
    • Función 'input()': Lee entrada del usuario desde la consola.
    • Función 'print()': Muestra salida en la consola.
    • Ejemplo de entrada/salida estándar:
      • nombre = input("Ingresa tu nombre: ") print(f"Hola, {nombre}!")
  5. Manejo de excepciones en archivos:
    • Es importante manejar posibles errores al trabajar con archivos.
    • Se usan bloques try-except para capturar y manejar excepciones.
    • Ejemplo de manejo de excepciones:
      • try: with open('archivo_inexistente.txt', 'r') as archivo: contenido = archivo.read() except FileNotFoundError: print("El archivo no existe.") except IOError: print("Error de entrada/salida al leer el archivo.")
Bibliotecas y frameworks populares
  1. NumPy:
    • Biblioteca fundamental para computación científica en Python.
    • Proporciona soporte para grandes arrays y matrices multidimensionales.
    • Incluye una amplia colección de funciones matemáticas de alto nivel.
    • Ejemplo de uso de NumPy:
      • import numpy as np # Crear un array arr = np.array([1, 2, 3, 4, 5]) # Operaciones vectorizadas print(arr * 2) # [2 4 6 8 10] # Funciones matemáticas print(np.mean(arr)) # 3.0
  2. Pandas:
    • Biblioteca para manipulación y análisis de datos estructurados.
    • Ofrece estructuras de datos como DataFrame y Series.
    • Proporciona herramientas para leer/escribir datos en varios formatos.
    • Ejemplo de uso de Pandas:
      • import pandas as pd # Crear un DataFrame df = pd.DataFrame({ 'Nombre': ['Ana', 'Juan', 'María'], 'Edad': [25, 30, 28] }) # Operaciones básicas print(df['Edad'].mean()) # 27.666666666666668 # Filtrar datos print(df[df['Edad'] > 27])
  3. Django:
    • Framework de alto nivel para desarrollo web rápido y limpio.
    • Sigue el patrón de diseño Modelo-Vista-Controlador (MVC).
    • Incluye un ORM para interactuar con bases de datos.
    • Ejemplo básico de una vista en Django:
      • # views.py from django.http import HttpResponse def hola_mundo(request): return HttpResponse("Hola, mundo!") # urls.py from django.urls import path from . import views urlpatterns = [ path('hola/', views.hola_mundo, name='hola_mundo'), ]
  4. Flask:
    • Microframework ligero para desarrollo web.
    • Fácil de usar y flexible, ideal para aplicaciones pequeñas a medianas.
    • Altamente extensible con numerosas extensiones disponibles.
    • Ejemplo básico de una aplicación Flask:
      • from flask import Flask app = Flask(__name__) @app.route('/') def hola_mundo(): return '¡Hola, mundo!' if __name__ == '__main__': app.run(debug=True)
